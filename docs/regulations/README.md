# 财务报告规范参考文档

本文档收集了上市公司财务报告（年报）的规范信息，用于提升PDF提取准确率。

---

## 一、规范来源

### 1. 证监会规范
- 《公开发行证券的公司信息披露内容与格式准则第2号——年度报告的内容与格式》
- 《公开发行证券的公司信息披露编报规则第15号——财务报告的一般规定》

### 2. 财政部企业会计准则
- **CAS 30** - 财务报表列报
- **CAS 31** - 现金流量表
- **CAS 32** - 中期财务报告

## 四、常见问题

### 问题1: 600000浦发银行现金流量表提取不完整
**原因**: PDF表格列标题被截断，导致科目名称不完整
**解决**: 
1. 添加更多变体匹配模式
2. 检查PDF解析器的跨页合并逻辑

### 问题2: 筹资活动产生被截断为"筹资活动产生"
**解决**: 在KEY_ITEM_PATTERNS中添加 r"筹资活动产生" 作为fallback模式
