# CF/IS 匹配率提升方案

**日期**: 2026-06-04
**状态**: 实施中
**关联**: [[2026-06-04-sina-rds-cleaning-pipeline-design]]

## 诊断

### CF (65.24%)
RDS CF 含直接法+间接法项目。Sina CF 只有直接法。以 600519 为例：
- RDS CF 54项，Sina CF 仅有 23项 (43%)
- 缺失的 29项中：2项在 IS (间接法)，27项完全不存在 (间接法调节项)
- **结论**: 不是别名问题，是表结构不同。需按直接法子集测量。

### IS (87.66%)
24个漏配项：名称变体 (管理费用/销售费用/财务费用/少数股东损益/归属于母公司)、金融特化 (已赚保费/退保金/赔付支出)。

## 方案

### CF: 直间分离 + 智能排除
1. 在 `learn_sina_aliases.py` 中新增 `learn_cf_direct_items()` — 从 RDS CF 中筛出 Sina CF 实际存在的项
2. 写 `rules/cf_direct_items.yaml` — 每只股票的 RDS CF 直接法项目清单
3. comparator 在计算 CF 匹配率时，只考虑"存在于 Sina CF"的项目作为 gt_items
4. 也检查跨表引用：CF 中 `净利润`/`财务费用`/`资产减值损失` 从 Sina IS 获取

### IS: 补全别名
1. 新增 manual_aliases to `sina_aliases_2019_2022.income_statement`:
   - 管理费用 / 销售费用 / 财务费用 变体
   - 少数股东损益 / 归属于母公司 变体
   - 信用减值损失（2019/2020格式差异）
2. 在 learn 脚本中把名字相似度阈值降到 0.3 (容纳更多中文变体)

### 目标
| 报表 | 当前 | 目标 |
|------|------|------|
| CF (直接法) | 65.24% | ≥ 90% |
| IS | 87.66% | ≥ 95% |
